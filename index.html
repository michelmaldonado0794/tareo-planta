Instrucciones para Administrar la Aplicación

Para actualizar las WOs o la URL del Webhook (Power Automate), debes modificar el archivo config.js.

  1. Abre la carpeta APP TAREO en tu computadora.
  2. Busca el archivo llamado config.js y ábrelo con el Bloc de notas.
  3. Pega tu nueva lista de WOs desde Excel entre las comillas invertidas.
  4. Guarda el archivo.
  5. Actualiza la página web para ver los cambios.
